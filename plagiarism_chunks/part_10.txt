The project is primarily divided into the following five core modules: 7.3.1 User Authentication and Profile Module This module serves as the primary entry point for all users, acting as the foundation for the system's data-driven logic. It handles identity verification, secure access control, and the ingestion of physiological data required by the machine learning models. Registration and Login: Utilizes JSON Web Tokens (JWT) and bcrypt password hashing to authenticate users securely. The implementation ensures that raw credentials are computationally transformed into cryptographic hashes before being stored in the database. Data Ingestion and Management: Captures vital physiological metrics from the user, including age, weight, height, gender, fitness goals (e.g., Muscle Gain, Fat Loss), experience level, and available physical equipment. This data is rigorously validated on the backend before being stored in the MongoDB document structure. Session Management: Maintains secure, stateless sessions across the React frontend and Express backend. By utilizing token-based authentication, the module ensures users only have access to their own sensitive health and fitness data, adhering to fundamental data privacy principles. 7.3.2 Artificial Intelligence Workout Generation Module This module acts as the core mathematical engine of the platform, responsible for transforming raw user metrics into structured, actionable fitness plans. Inference Engine: Transmits the user's profile data via REST API to the Python FastAPI server. The module normalizes and encodes the categorical user data (such as gender and goal) into numerical formats suitable for machine learning inference. Machine Learning Integration: Feeds the preprocessed variables into pre-trained Extreme Gradient Boosting (XGBoost) regression models. XGBoost was selected for its high execution speed and ability to capture complex, non-linear relationships within tabular fitness data. Dynamic Plan Construction: The module mathematically predicts optimal exercise selections, target sets, and repetition counts. It synthesizes these individual predictions into a comprehensive, seven-day workout schedule, formatted as a JSON payload, and returns it to the client. 7.3.3 Artificial Intelligence Nutrition Module To support the physical workouts outputted by the generation module, this module calculates the metabolic requirements necessary to achieve the user's fitness objectives. Macronutrient Calculation: Computes the precise daily caloric targets utilizing established metabolic formulas. It calculates the necessary ratio of proteins, carbohydrates, and fats required to support the user's stated goal, whether that involves operating at a caloric deficit for weight reduction or a surplus for muscle hypertrophy. Meal Recommendation Logic: Generates meal recommendations based on the calculated caloric and macronutrient requirements. By mapping the AI-predicted numerical targets to a deterministic, rule-based selection algorithm, the module ensures that the assigned dietary items strictly adhere to the mathematical boundaries established by the user's physical profile. 7.3.4 Real-Time Computer Vision and Tracking Module This module represents the interactive component of the platform, enabling live spatial feedback during active workout sessions without requiring external hardware beyond a standard webcam. Camera Integration: Interfaces directly with the user's device webcam through the browser using asynchronous JavaScript APIs, establishing a continuous video stream. Pose Estimation: Deploys Google MediaPipe via WebAssembly to detect 33 distinct 3D spatial landmarks on the human body in real time. The model operates continuously, drawing a digital skeleton over the user's physical form. Kinematic Analysis: Calculates specific joint angles (such as the interior angle of the elbow or the flexion of the knee) using trigonometric functions applied to the spatial coordinates. This allows the system to automatically count repetitions and provide instantaneous, programmatic feedback regarding exercise form. All processing occurs locally on the client's device to preserve privacy and minimize transmission latency. 7.3.5 Dashboard and Analytics Module This module acts as the central hub for the user interface, synthesizing the complex data structures returned by the backend engines into a coherent, easily navigable visual format. Progress Tracking: Aggregates database records to display daily completion status, active user streaks, and historical consistency metrics, displaying daily completion status, active user streaks, and historical consistency metrics to encourage regular platform usage. Plan Management: Allows users to interact dynamically with their AI-generated plans. Users can view specific exercise execution details and request algorithmic "swaps" if a particular exercise requires unavailable equipment or causes discomfort. Data Visualization: Utilizes Tailwind CSS and structured React components to present complex numerical data—such as workout statistics and nutritional summaries—in a clean, easily interpretable graphical interface, ensuring the application remains accessible to users lacking deep technical or fitness knowledge. 7.4 Algorithm Implementation The intelligence and interactive capabilities of the Elevate platform are driven by two primary algorithms. The first is a machine learning inference algorithm responsible for personalized plan generation, and the second is a trigonometric algorithm responsible for real-time computer vision analysis. 7.4.1 XGBoost Inference Algorithm (Workout & Nutrition Prediction) Extreme Gradient Boosting (XGBoost) is a supervised learning algorithm based on the principle of decision tree ensembles. Instead of relying on a single complex tree, XGBoost builds a sequence of shallow "weak learner" trees. Each successive tree is explicitly designed to minimize the residual errors (the difference between the predicted value and actual value) of the previous trees using gradient descent optimization. For structured tabular datasets such as age, weight, height, and fitness goals, XGBoost often provides excellent predictive performance while maintaining low computational overhead.
